# LOF基金溢价率计算工具

一个用于获取和分析LOF基金溢价率的Python工具。

## 功能

- 使用akshare获取所有LOF基金的实时溢价率数据
- 筛选有溢价的基金
- 支持特别关注基金列表
- 按溢价率排序显示TOP5基金

## 安装

```bash
pip install akshare pandas
```

## 运行

```bash
python main.py
```

## 部署

这个代码实现的是coze的插件，代码粘贴到coze上即可使用，具体请参考coze文档。main函数是为了方便本地测试。
